# Emulator Tools

This folder contains Python tools for assembly parsing, emulation, and GUI workflows.

## Files
- `assembly.py`: Assembly loader/parser.
- `emulator.py`: RISC-V emulator core.
- `riscv_emulator.py`: CLI entrypoint for emulator runs.
- `gui.py`: Tkinter GUI frontend for emulator inspection.
- `cli.py`: Additional command-line utility support.

## Notes
- Activate the project virtual environment before running tool scripts.
- Keep tool-specific helper modules in this folder.
