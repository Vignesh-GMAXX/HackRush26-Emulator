# part1 Simulator Core

This folder contains the core simulator implementation and supporting tooling.

## Subfolders
- `src/`: C source files for simulator logic.
- `include/`: Header files.
- `scenarios/`: Input scenario JSON files.
- `bin/`: Built executable output.
- `asm/`: RISC-V assembly kernels and examples.
- `emulator/`: Python-based emulator and GUI tools.

## Build
Use the top-level scripts or run `make` from this folder.
